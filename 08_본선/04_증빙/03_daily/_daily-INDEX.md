# _daily-INDEX

> **데일리 업무 노트 허브.** 여러 세션(사람+AI, 병렬 4개 내외)이 같은 볼트에서 일하므로, 하루 단위 **조율 보드**가 필요하다. `session-log`(세션별 append 스트림)와 역할이 다르다:
> - **session-log** = 세션 종료 시 각자 남기는 시계열 기록(과거).
> - **데일리 노트** = 그날의 **현재 상태 롤업 + 열린 핸드오프 + 대기 결정**(지금·조율용).
>
> 규약: 하루 첫 세션이 [[_daily-template]]로 그날 노트를 만들고, 이후 세션은 자기 작업·주의사항을 **해당 날짜 노트에 append**한다. 파일명 = `YYYY-MM-DD.md`(이 폴더에만). 루트에 생기면 여기로 옮긴다.
