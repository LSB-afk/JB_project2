# JB우리캐피탈 업무처리 기술서 — 여신 라이프사이클 × 조직 × 통제

> 조직은 [[우리캐피탈-조직도]], 상품·채널 원문은 [[우리캐피탈-사이트-IA분석]] 기준.
> 사이트에서 확인된 사실(상품개발 프로세스·판매원칙·소비자보호 체계·조직 편제)과 (해석)으로 보완한 업무 흐름을 구분 표기.

## 1. 여신 라이프사이클 — 엔드투엔드 처리 흐름

### ① 상품 기획·개발 (사이트 확인: 상품개발 프로세스 4단계, `/csm/JCSMPRC0001.do`)

| 단계 | 사이트 명시 활동 | 담당 조직 (해석) |
| --- | --- | --- |
| 기획 | 시장 트렌드 분석 · 불완전판매 점검 | PL기획팀(개인) · 중고차기획팀/신차금융팀(오토) |
| 개발 | 약관·상품 설명서 점검 · 소비자보호 체크리스트 점검 | 상품 기획팀 + 소비자보호팀 협의 |
| 판매 | 상품판매 전략 수립 · **제휴·위탁업체 교육** · 홈페이지 상품안내 | 영업본부 + Digital금융팀 |
| 사후관리 | 상품관련 내부·외부 **민원 모니터** | 소비자보호팀 → 주무부서 피드백 |

- 사이트 명시: 신상품이 "금융소비자의 권익 침해 가능성이 있는 경우 주무부서에 신상품 출시 및 마케팅 중단 등의 요구가 가능"한 내부지침 운영 → **소비자보호 조직이 상품 출시를 멈출 수 있는 권한**을 가짐.

### ② 모집·판매 — 채널 3원화 + 제휴망 (해석, 조직·사이트 기능으로 뒷받침)

| 채널 | 실체 | 처리 방식 |
| --- | --- | --- |
| 비대면 | Digital금융팀 · 홈페이지/앱 | "차량번호 하나로", 간편 견적조회(리스·렌터카), 원클릭신청 |
| 전화 | TM센터 | 아웃/인바운드 상담 판매 |
| 대면 | 개인금융센터 · 오토 7개 지점/사무소 | 고액·복잡 건, 딜러 현장 |
| 제휴 | 신차·중고차 딜러망 (상품개발 프로세스의 "제휴·위탁업체 교육"으로 존재 확인) | 판매점 경유 오토금융 — 여전사 오토금융의 주력 유입 경로 |

### ③ 신청·서류 접수 (사이트 확인: 원클릭신청)

- **온라인서류등록**: 구비서류 비대면 제출.
- **전자약정**: 계약 체결 자체를 전자화.
- 내 금융관리 > **내 진행현황**: 고객이 심사 진행 상태를 직접 조회 — 처리 상태가 단계화되어 있다는 증거.

### ④ 심사 (조직 확인: 심사본부 / 위원회)

- 리테일: 리테일심사팀 — (해석) CSS(리스크모형팀 개발) 기반 자동 승인 + 예외 수기 심사.
- 기업: 기업심사 3개 팀 — (해석) 딜 규모별·업종별 분담, 재무·담보·프로젝트 심사.
- 금리·한도 결정: **금리심사위원회** · **여신위원회** (조직도 확인) — 위원회 결재 체계.
- (해석) 여전사 특성상 은행 거절 차주 유입이 있어 리스크 프라이싱(고금리·고한도 스펙트럼)이 심사의 핵심 기술.

### ⑤ 실행·자산 운영

- 오토운영팀·집중화지원팀: 실행 서류·기표·딜러 정산 집중 처리 (해석: "집중화지원" = 백오피스 집중화).
- 오토자산관리팀: 리스·렌터카 **차량 자산** 관리 — 내 금융관리 > **차량관리** 메뉴와 대응 (저당 설정·해지, 리스 만기 처리 등).
- 결제/발급: 납입 스케줄·증명서 발급 셀프서비스.

### ⑥ 사후관리 — 제품 핵심 접점 ★

- **여신관리본부**: 여신관리기획팀(전략·세그먼트) + 여신관리운영팀 + **전국 8개 센터**(실행) + 여신관리지원센터.
- 신용관리팀(리스크관리본부): (해석) 차주 신용 모니터링·조기경보 기준 운영.
- (해석) 처리 흐름: 연체 전 징후 감지 → 접촉(안내·상환 유예·대환 제안) → 연체 시 채권관리(센터) → 회수 불능 시 상각·NPL 이관.
- **콘솔 서사 연결**: 이 단계가 [[제품-정의]]의 "여신 사후관리" 페인포인트. 상환위험 분류 에이전트(히어로 owner)가 붙는 자리.

### ⑦ 부실채권(NPL) 처리

- NPL금융1·2팀(기업금융본부): (해석) 자사 부실채권 처리 + **타사 NPL 투자**를 겸하는 수익 사업. 사후관리 품질이 곧 NPL 원가.

### ⑧ 민원·피드백 루프 (사이트 확인: 소비자보호 체계 3단계)

- 개발단계 → 판매단계 → 사후관리 전 단계에 소비자보호 내부지침.
- "원인을 분석하고 제도를 개선하는 **민원 피드백 시스템** 구축·운영" + **소비자보호 이행 여부를 부서 성과평가(KPI)에 반영** (사이트 명시).

## 2. 사업영역별 처리 방식 (사업영역 페이지 + 상품 페이지)

| 영역 | 처리 특징 | 근거 문구 (사이트) |
| --- | --- | --- |
| **자동차금융** | 딜러 접점 + 견적조회 → 신청 → 차량 담보/소유권 구조(할부=소유 이전, 리스=회사 소유, 렌트=대여) | "고객의 특별한 상황에 맞게 맞춤형 자동차 금융 상품" |
| **개인금융** | 비대면·속도 우선. 담보(차량) 또는 신용, 균등 분할납입 구조 | "급하게 자금이 필요한 개인", "간편한 구비서류 및 절차", "균등 분할납입으로 만기 목돈 상환 부담 경감" |
| **부동산금융** | 주담대 — LTV 기반, 분할 상환 | "목돈에 대한 부담" 해소, "분할 상환 가능" |
| **기업금융·IB** | 딜 단위 심사·위원회 승인·신디케이션 (해석). 일반대출~PF·메자닌·PEF·M&A·구조화·신기술·팩토링 | "기업의 재무 구조 개선 지원" |

## 3. 거버넌스·통제 체계 (사이트 확인)

### 3-1. 금융소비자보호 조직 (`/csm/JCSMOGN0001.do`)

```
대표이사 ─ 금융소비자보호 내부통제위원회
   │
금융소비자보호 총괄책임자(CCO) ─ 금융소비자피해보상 심의위원회
   │
소비자보호팀 · 고객만족센터
```

### 3-2. 6대 판매원칙 (`/csm/JCSMRUL0001.do` — 금소법 대응)

| 원칙 | 사이트 서술 |
| --- | --- |
| 적합성 원칙 | 소비자에게 부적합한 금융상품 권유 금지 |
| 적정성 원칙 | 구매하려는 상품이 재정상태에 비추어 부적정 시 고지·확인 |
| 설명의무 | 계약체결 권유·설명 요청 시 중요사항 설명 |
| 불공정영업행위 금지 | 우월적 지위 이용 권익 침해 금지 |
| 부당권유행위 금지 | 오인 우려 허위 사실 고지 금지 |
| 허위·과장광고 금지 | 광고 필수 포함 사항·금지행위 준수 |

→ **콘솔 연결**: 준법 검토 에이전트(L3)의 체크리스트 원형. RM 에이전트의 모든 고객 대상 액션 초안은 이 6원칙 필터를 통과해야 한다는 시나리오 근거.

### 3-3. 위원회 결재 체계 (조직도 확인 8개)

집행위원회(경영) · 여신위원회 / 금리심사위원회(여신 승인·프라이싱) · 리스크관리운영위원회 · 전산운영위원회 / 정보보호위원회(IT·보안) · 내부통제협의회 · 인사위원회
→ (해석) **L0~L4 승인 라우팅의 현실 대응물**: 콘솔의 승인 게이트 = 위원회·전결 체계의 디지털 미러.
