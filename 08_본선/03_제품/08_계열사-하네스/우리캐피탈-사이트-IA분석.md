> **수집 방법론(투명성)**: 2026-07-02~03 정적 fetch + 텍스트 추출. 회사소개(`/cmp`)·금융소비자보호(`/csm`) 페이지는 본문까지 확보. **금융상품(`/fin`) 상세 본문은 동적(XHR) 렌더링이라 미확보** — 상품별 상세는 페이지 메타(타이틀·설명·키워드)와 사업영역 페이지 서술로 분석했고, 한도·금리 등 세부 스펙은 인용하지 않는다(추가 확보 시 갱신).

│   ├── 자동차금융: 중고차대출 · 자동차리스 · 장기렌터카

## 4. 금융소비자보호 섹션 (본문 확보 완료)

- **헌장·규정·공시**: 금소법 체계 공시(`/csm/JCSMANC*`), 상품공시실.
- **조직**: 대표이사 → 내부통제위원회 / CCO → 피해보상 심의위원회 / 소비자보호팀·고객만족센터.
- **체계 3단계**: 개발(출시 중단 요구권) → 판매(정보 제공 지침) → 사후관리(민원 피드백 시스템 + **부서 KPI 반영**).
- **6대 판매원칙**: 적합성·적정성·설명의무·불공정영업금지·부당권유금지·허위과장광고금지 → 상세는 [[우리캐피탈-업무처리-기술]] §3-2.
- **금융사기피해예방** 메뉴 존재 → 보이스피싱 도메인과 직결 ([[제품-정의]] 페인포인트).
- **전자민원**: 민원 신청 채널 온라인화.

## 5. UX·톤 관찰 (콘솔 디자인 참고)

- 마이크로카피가 구어체 존대: *"시스템 점검 및 서비스 이용에 필요한 내용을 공지해드려요"*, *"친절하게 상담해드려요"* — JB 계열 공통 "~해요" 톤. → 콘솔 고객 대상 문구 초안 톤 참고.
- 상담 CTA가 전 상품 페이지 공통: "상담원 전화연결" / "빠른 상담신청" / "견적 내기" — 비대면과 휴먼 상담의 하이브리드.
- 로딩 문구 "신분증을 확인하고 있어요" — eKYC 흐름 존재 신호.
