## 제품·사업 적용 시사점

- **규모 근거**: 순익 7,104억·원화대출 +7.7%·RWA +3.9% 조합 = 자본효율 중심 선별 영업이 RM KPI 핵심. JB LocalGuard OS의 "승인 게이트" 설계를 정당화하는 정량 근거로 직접 인용 가능.
- **은행별 건전성 편차**: 전북은행 연체율 1.46% > 광주은행 1.01% — 두 은행의 EWS 임계값을 달리 설정해야 함. [미검증] 상태이지만 전북은행 공식 경영목표 페이지(1차)에서 1.46%는 확인됨.
- **캐피탈 분기 변동성**: 연체율 2025Q1 2.82% → 연말 2.21% → 2026Q1 2.83% 반등 패턴 = 월·주 단위 EWS 필요. 은행보다 짧은 모니터링 주기를 제품 설계에 반영.
- **소상공인 대환대출**: 2023~2024 누계 1,788억·44.7% 점유율은 RM AI가 다룰 정책대환 풀의 규모 근거. 단, "2024년 687억"으로 제출 문서에 인용하면 연도 오류 — 아래 canon 갱신 필요.
- **JB자산운용 AUM 6.35조**: RM이 발굴한 법인·자산가 고객을 IB/대체투자로 연결하는 "그룹 시너지 추천" 기능의 근거 수치로 활용 가능.

2. **광주은행 총자산·자기자본 절대액**: 이번 검증에서도 DART/IR 직접표 미확보. 수익성 지표만 KIND 공시로 확인.
3. **JB우리캐피탈·JB인베스트먼트 총자산·AUM**: 역시 DART PDF 직접 확인 미완.

**핸드오프 요약 (6블록)**

1. **검증 통과**: 순익 7,104억·ROE 12.4%·ROA 1.04%·CET1 12.58% — 제출 문서 그대로 인용 가능.
2. **연도 오류 즉시 수정**: _canon §10 "2024년 687억" → "2023년 687억; 2023~2024 누계 1,788억(44.7%)".
3. **총자산 73.1조 보류**: D3a 1차 주장이지만 WebFetch 직접 미확인. 제출 문서 인용 전 DART 사업보고서 재무상태표 재확인 필요. 임시 표기 "FY2025 약 73조(확인 중)" 권장.
4. **[미검증] 다수**: 광주은행 총자산·자기자본, 캐피탈 포트폴리오 비중, 지역 점유율, 분기별 연체율 — 실무 참고 가능하나 심사 인용 전 DART PDF 필수.
5. **SOHO 잔액 공백**: D3a에서 확보 실패. 사업보고서 별도 재조사 필요.
6. **_canon 갱신 필요 항목 2건**: (a) 대환대출 연도 오류 즉시 수정, (b) 총자산 DART 재확인 후 66~67조 범위 표기 갱신.
